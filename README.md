# Next.js Auth Template

Production-ready Next.js template with WorkOS authentication.

## Quick Start

```bash
npm install
cp .env.example .env.local
# Add WorkOS credentials to .env.local
npm run dev
```

## Features

- WorkOS Authentication
- Protected routes with middleware
- React Query
- Tailwind CSS + shadcn/ui
- TypeScript

## Setup

1. Get WorkOS credentials from dashboard.workos.com
2. Add to .env.local
3. Run npm run dev
